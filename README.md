# ANPR & Automated Traffic Control System

This project implements Automatic Number Plate Recognition (ANPR) and a basic Automated Traffic Control System (ATCC) using OpenCV and OCR techniques. The system detects vehicles from images or video, extracts number plates, recognizes characters, and supports automated traffic monitoring.
